Example: Quadratically Constrained Quadratic Optimization

\[ \max \bigg\{x^{\intercal} A_0 x : \]
\[ x^{\intercal} A_1x = c_1,\dots, x^{\intercal} A_k x = c_k \] \[x\in S^{n-1}\]

Here, each $A_i$ is a symmetric matrix, and $S^{n-1}$ denotes the unit sphere in $\R^n$.

Optimization

Example: Linear Inverse Eigenvalue Problems

\[ \max \bigg\{\tr(A_0 X) : \]
\[ \tr(A_1 X) = c_1,\dots, \tr(A_k X) = c_k \] \[X\in M_{\lambda}\]
\[ \bigg\} \]

For $\lambda \in \R^n$, $M_{\lambda}$ is the set of symmetric matrices with eigenvalues $\lambda_1, \dots, \lambda_n$.

QCQPs over the sphere are the special case of $\lambda= (1,0,\dots,0)$.

Here, each $A_i$ is an $n\times m$ matrix, and $\St^{n,m} = \{X \in \R^{n \times m} : X^{\intercal}X = I\}$.

Convex Relaxations

Convex relaxations can usually be tractably solved, and they can prove upper bounds showing that no solution can achieve a certain objective value.

In general, these convex relaxations give upper bounds which are loose.

Question: When can we find a tight convex relaxation for these types of problems?

Example: Brickman's Theorem

If we have one constraint in a QCQP over the sphere, the natural convex relaxation is tight.

\[ \max \bigg\{x^{\intercal} A_0 x : \]
\[ x^{\intercal}A_1x = c_1 \]
\[x \in S^{n-1}\]
\[ \bigg\}= \]
\[ \max \bigg\{\langle A_0, X \rangle : \]
\[ \langle A_1, X \rangle = c_1 \]
\[\tr(X) = 1, X \succeq 0\]
\[ \bigg\} \]

One way to see this is to use Brickman's theorem that $\{(x^{\intercal}A_0x,x^{\intercal}A_1x) : x \in S^{n-1} \} \subseteq \R^2$ is convex.

(Generalized) Lagrangians

(Generalized) Lagrangians

Given a function $f : \mathcal{X} \rightarrow \R^{k+1}$, we can define a (generalized) Lagrangian as a function of $x \in \mathcal{X}$ and $t \in \R^{k+1}$: \[ \mathcal{L}_f(x,t) := \langle t, f(x)\rangle = \sum_{i=0}^{k} t_i f_i(x). \]

We can think of $t$ as parameterizing a family of objective functions on $\mathcal{X}$: \[ \ell_t(x) = \mathcal{L}_f(x,t). \]

For each value of $t \in \R^{k+1}$, we get a different maximization problem, $\max \{\ell_t(x) : x\in \mathcal{X}\}$.

Definitions

A maximizing section associated to a function $f : \mathcal{X} \rightarrow \R^{k+1}$ is a function $D : \R_{\ge 0}\times \R^{k} \rightarrow \mathcal{X}$ so that for every $t \in \R_{\ge 0}\times \R^{k}$, $D(t)$ is a maximizer of the function $\ell_t(x) = \mathcal{L}(x,t)$.

A function $f : \mathcal{X}\rightarrow \R^{k+1}$ is continuously maximized if there is a maximizing section which is continuous as a function of $t$.

Theorem

If $\mathcal{X}$ is a compact metric space, and $f : \mathcal{X}\rightarrow \R^{k+1}$ is continuously maximized , then the function $V_f(c)$ is concave.

\[ V_f(c) := \max \bigg\{f_0(x) : \]
\[ f_1(x) = c_1,\dots,f_k(x) = c_k \] \[x\in \mathcal{X}\]
\[ \bigg\} \]

Example: Brickman's Theorem

Consider once again the one constraint case of the QCQP:

\[ V_f(c)= \]
\[ \max \bigg\{x^{\intercal} A_0 x : \]
\[ x^{\intercal}A_1x = c_1 \] \[x \in S^{n-1}\]
\[ \bigg\} \]

The Lagrangian in this case is $x^{\intercal} (t_0 A_0 + t_1 A_1) x$: a maximizer of the Lagrangian on the sphere is an eigenvector of $t_0 A_0 + t_1 A_1$ associated to the largest eigenvalue.

When is this continuous?

Example: Brickman's Theorem

The Von Neumann-Wigner noncrossing theorem states that for almost all pairs of symmetric matrices $(A_0, A_1)$, every nonzero matrix in their span contains no eigenvalues with multiplicity greater than 1.

The eigenvectors of a matrix are continuous outside of the set of matrices with repeated eigenvalues.

So for almost all pairs $(A_0, A_1)$ the function $f(x) = (x^{\intercal} A_0 x, x^{\intercal} A_1 x)$ is continuously maximized.

Combining this with some elementary homotopy theory gives Brickman's theorem that the image of the sphere under any such pair of quadratic functions is a convex set.

Other examples

Higher Dimensional Examples

Concavity does not hold in general for larger numbers of constraints (for good reason!)

Are there any higher dimensional examples?

Higher Dimensional Examples

A vector subspace of the vector space of $n\times n$ symmetric matrices is said to be $n-w$-weakly noncrossing if the $w$ largest eigenvalues of every nonzero matrix in that subspace are nondegenerate (i.e. have one dimensional eigenspaces).

A 0-weakly noncrossing subspace is simply called noncrossing.

Almost every 2 dimensional subspace is noncrossing by Von Neumann-Wigner.

Higher Dimensional Examples

Theorem (Friedland, Robbin, Sylvester 1984)

A $d$ dimensional noncrossing subspace of the space of $n\times n$ symmetric matrices exists if and only if $d$ is at most $\sigma(n)$ where \[ \sigma(n) = \begin{cases} 2 \text{ if }n \not \in \{-1,0,1\} \mod 8\\ \rho(4n) \text{ otherwise} \end{cases}, \] where $\rho(n)$ is the Radon-Hurwitz number: if $n = 2^{4a + b}c$, where $a,b,c$ are integers with $c$ odd and $0 \le b \le 3$, then $\rho(n) = 8a + 2^b$.

$n-1$-weakly noncrossing subspaces of dimension $d$ exist whenever $d$ is at most $n$ (Friedland-Loewy 1976).

Higher Dimensional Examples

Theorem

If $A_0, \dots, A_k$ span a $w$-weakly noncrossing subspace, and $\lambda_i = 0$ for $i \ge n-w$, then $f(X) = (\tr(A_0X), \dots, \tr(A_kX)) $ is continuously maximized on $M_{\lambda}$.

Convexity in the case of QCQPs over the sphere was proven in (Gutkin-Jonckheere-Karow 2004)

Analogous results hold for the singular value analogues.

Unbalanced Procrustes Problems

Unbalanced Procrustes Problems

Given a pair $U \in \R^{n \times d}$, and $V \in \R^{m \times d}$ with $n > m$, we want to minimize \[ \|U X - V\|^2_2 \] over $X \in \St^{n,m}$ satisfies $X^{\intercal}X = I$.

Both the natural SDP relaxations for this problem and Riemannian gradient descent with random initialization fail to find the global minimum for this problem.

Unbalanced Procrustes Problems

We can rename $A = -VV^{\intercal} \in \R^{n\times n}_{sym}$ and $B = 2 V^{\intercal}W \in \R^{n \times m}$, and rewrite the problem as \[ \max \{\tr(A XX^{\intercal}) + \tr(B^{\intercal}X) : X \in \St^{n,m}\}. \]

Conjecture

For all $n, m$ and all but a set of measure zero of pairs $A \in \R^{n\times n}_{sym}$ and $B \in \R^{n \times m}$, the function \[ f(X) = (\tr(A XX^{\intercal}), \tr(B^{\intercal}X)) \] is continuously maximized on $\St^{n,m}$.

We can show this when $(n, m) = (3,2)$ or when $m = 1$, but not in general.

The Projection Profile

We can think of the two terms in the objective as separate entities: i.e. look at the set of possible pairs \[ P_{A,B} = \{(\langle A, XX^{\intercal} \rangle, \langle B, X\rangle) : X \in \St^{n,m}\} \subseteq \R^2. \]

Relationship with Homotopy Continuation

Can we minimize $\ell_t(x) = \mathcal{L}(x,t)$ assuming that $f$ is continuously maximized?

Homotopy continuation is a method, usually used to solve polynomial systems, that at a high level starts with some initial system and then varying the parameters of that system slowly while tracking the solutions to the system over time.

Relationship with Homotopy Continuation

Can we minimize $\ell_t(x) = \mathcal{L}(x,t)$ assuming that $f$ is continuously maximized?

Along the lines of homotopy continuation, we can think about the following algorithm:

  1. Input A Lagrangian $\mathcal{L}(x,t)$, an initial $t_0$, a maximizer $x_0$ of $\ell_{t_0}(x)$, and a target $t_1$.
  2. Choose time steps $t_{1/N}$, $t_{2/N}$, \dots, $t_{N-1 / N}$.
  3. For $i = 1, \dots, N$, starting at $x_{i-1/N}$, apply gradient descent until we arrive at an estimate $x_{i/N}$.

If the maximizer is Lipschitz as a function of $t$, and also, the local minima of $\ell_{t_i}$ are well separated, we expect this to track the optimum correctly.
